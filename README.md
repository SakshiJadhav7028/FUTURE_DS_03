# FUTURE_DS_03
Student Feedback And Satisfaction Analysis

Project Description
This project analyzes student feedback survey data to measure satisfaction levels and identify areas of improvement using Python data analysis techniques.

-Tools Used
  1)Python
  2)pandas
  3)matplotlib
  4)Google Colab

-Features
   Data cleaning and preprocessing
   Rating-based sentiment analysis
   Data visualization
   Insight generation and recommendations

-Output
  1)Average rating analysis
  2)Sentiment classification (Positive, Neutral, Negative)
  3)Visual charts and graphs
